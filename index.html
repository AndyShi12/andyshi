CPU Architecture & Design

Created a CPU based on the MIPS architecture. This semester long project builds upon one design; starting with a singlecycle MIPS implementation (ALU, Control Unit, Datapaths), followed by a 5 stage pipeline (Fetch / Decode / Execute / Memory / Write), Hazard Unit, Forwarding Unit, L1 instruction and data cache, and ultimately combining everything with MSI coherence and LL/SC Synchronization into a single multicore design.
